확률과 통계 · Hogg (§1.5–1.9)

2 확률변수와 분포

“확률은 사건에, 확률변수는 숫자에” — 동전과 주사위의 우연을 수직선 위의 함수로 번역하는 순간, 해석학의 적분과 급수가 통째로 확률 계산기로 바뀐다.

1장의 확률공간 $(\Omega, \mathcal{F}, P)$는 강력하지만 불편하다 — 표본점 $\omega$가 “앞앞뒤”, “빨간 공”처럼 숫자가 아니기 때문이다. 확률변수는 이 표본점 하나하나에 실수 하나를 붙여, 우연을 수직선 위로 끌어내리는 번역기다. 번역이 끝나면 “동전 세 번에 앞면이 나올 확률”은 “$X$가 특정 값을 가질 확률”이 되고, 기댓값·분산·적률생성함수라는 세 가지 요약 장치가 분포 전체를 압축한다. 이 장은 그 번역의 문법 — 분포함수, 기댓값, 적률생성함수, 그리고 변환 — 을 세운다.

1. 확률변수와 분포함수 — 표본공간에서 수직선으로

직관부터. 확률변수 $X$는 “측정”이다. 주사위를 굴리면($\omega$) 눈금을 읽고($X(\omega)$), 사람을 뽑으면 키를 잰다. 일단 숫자로 옮겨지면 우리가 진짜 알고 싶은 것은 $\omega$가 아니라 “$X$가 어디에 얼마의 확률로 떨어지는가” — 이 정보 전체를 한 함수에 담은 것이 누적분포함수다.

정의 · 확률변수와 분포함수 random variable, CDF (호그 §1.5–1.7)

확률변수는 함수 $X : \Omega \to \mathbb{R}$이다 (모든 $x$에 대해 $\{\omega : X(\omega) \le x\}$이 사건이 되도록 하는 가측성 조건을 붙인다). 그 누적분포함수(CDF)는

$$F(x) = P(X \le x) = P(\{\omega : X(\omega) \le x\})$$

로 정의한다. $F$ 하나가 $X$의 확률적 성질을 완전히 결정한다 — 이후 모든 계산의 원천이다.

이산형: $X$의 값이 유한 또는 가산이면, 확률질량함수(pmf) $p(x) = P(X = x)$가 있고 $\sum_x p(x) = 1$. $F$는 계단함수이며 각 도약의 높이가 바로 $p(x)$다.

연속형: 음이 아닌 확률밀도함수(pdf) $f$가 있어 $$F(x) = \int_{-\infty}^{x} f(t)\,dt, \qquad \int_{-\infty}^{\infty} f(t)\,dt = 1,$$ 이고 (해석학의 적분이 그대로 무대다) 구간 확률은 넓이로 읽는다: $$P(a \lt X \lt b) = \int_a^b f(x)\,dx.$$ 미적분학의 기본정리로 $F' = f$ (연속점에서).

정리 · 분포함수의 성질 properties of a CDF (호그 §1.5)

어떤 함수가 CDF가 될 수 있는가? 다음 세 성질이 정확한 관문이다.

① 단조증가: $x_1 \lt x_2 \Rightarrow F(x_1) \le F(x_2)$ (더 넓은 구간이 확률을 더 담는다).
② 양 끝 극한: $\displaystyle\lim_{x\to-\infty}F(x) = 0$, $\displaystyle\lim_{x\to+\infty}F(x) = 1$.
③ 우연속: $\displaystyle\lim_{x\to a^+}F(x) = F(a)$ (사건이 $\le a$로 등호를 포함하기 때문 — 왼쪽이 아니라 오른쪽에서 이어진다).

결정적 공식 두 개:

$$P(a \lt X \le b) = F(b) - F(a), \qquad P(X = a) = F(a) - F(a^-)$$

즉 점질량은 $F$의 도약 폭이다. 연속형은 $F$가 연속이라 도약이 없어 $P(X = a) = 0$ — 이 한 줄이 아래 “흔한 오해”의 씨앗이다.

예제 · 이산과 연속, 나란히

(a) 공정한 주사위(이산). $X \in \{1,\dots,6\}$, $p(x) = \tfrac16$. $F$는 각 정수에서 $\tfrac16$씩 뛰는 여섯 계단. $P(2 \lt X \le 5) = F(5) - F(2) = \tfrac56 - \tfrac26 = \tfrac36 = \tfrac12$.

(b) 균등분포 $U(0,1)$(연속). $f(x) = 1$ ($0 \lt x \lt 1$), $F(x) = x$ (그 구간에서). $P(X = 0.3) = 0$이지만 $P(0.2 \lt X \lt 0.5) = 0.3$ — 한 점의 확률은 0인데 구간의 확률은 양수. 넓이의 관점에서 지극히 자연스럽다 (선분의 넓이는 0).

(c) 지수분포 $\mathrm{Exp}(\lambda)$. $f(x) = \lambda e^{-\lambda x}$ ($x \gt 0$), $F(x) = 1 - e^{-\lambda x}$. 대기시간의 표준 모형이며, $P(X \gt s+t \mid X \gt s) = P(X \gt t)$ (무기억성)이 이 $F$에서 곧바로 나온다.

2. 기댓값과 분산 — 분포를 두 숫자로 요약

분포 전체를 들고 다니기는 무겁다. 기댓값은 분포의 “무게중심” (값들을 확률로 가중평균한 위치), 분산은 그 중심 둘레로 얼마나 퍼졌는지다. 물리의 질량중심과 관성모멘트에 정확히 대응한다.

정의 · 기댓값과 분산 expectation and variance (호그 §1.8–1.9)

기댓값(평균) $\mu = E[X]$:

$$E[X] = \sum_x x\,p(x) \quad(\text{이산}), \qquad E[X] = \int_{-\infty}^{\infty} x\,f(x)\,dx \quad(\text{연속})$$

단, 합·적분이 절대수렴할 때만 존재한다 (조건수렴이면 재배열로 값이 흔들려 정의가 무의미 — 해석학 3장의 교훈).

무의식적 통계학자의 법칙(LOTUS): $Y = g(X)$의 분포를 새로 구할 필요 없이, $X$의 분포로 바로 계산한다.

$$E[g(X)] = \sum_x g(x)\,p(x) \quad\text{또는}\quad \int_{-\infty}^{\infty} g(x)\,f(x)\,dx.$$

분산과 표준편차: $g(x) = (x-\mu)^2$를 넣은 특수한 기댓값.

$$\operatorname{Var}(X) = E\big[(X-\mu)^2\big] = E[X^2] - \mu^2, \qquad \sigma = \sqrt{\operatorname{Var}(X)}.$$

오른쪽 등식(계산용 공식)은 아래 증명에서 유도한다.

선형성 — 실전에서 가장 자주 쓰는 두 규칙:

$$E[aX + b] = aE[X] + b, \qquad \operatorname{Var}(aX + b) = a^2\operatorname{Var}(X).$$

평균은 평행이동을 따라가지만, 분산은 평행이동에 무감각하고 ($b$는 퍼짐을 바꾸지 않는다) 축척에는 제곱으로 반응한다. 이 사실로 표준화 $Z = (X-\mu)/\sigma$가 항상 평균 $0$, 분산 $1$이 된다.

적률: $k$차 적률 $E[X^k]$, $k$차 중심적률 $E[(X-\mu)^k]$. 평균은 1차 적률, 분산은 2차 중심적률이다.

증명 — 분산의 계산 공식 $E[X^2]-\mu^2$은 제곱을 전개하고 선형성으로 상수를 밖으로 빼면 끝. 체비쇼프는 분산 적분에서 “먼 곳”만 남기는 한 줄 부등식.

계산 공식: $\mu = E[X]$는 상수이므로

$$E[(X-\mu)^2] = E[X^2 - 2\mu X + \mu^2] = E[X^2] - 2\mu\,E[X] + \mu^2 = E[X^2] - 2\mu^2 + \mu^2 = E[X^2] - \mu^2.$$

선형성: $E[aX+b] = \sum (ax+b)p(x) = a\sum x\,p(x) + b\sum p(x) = aE[X] + b$ (마지막에 $\sum p = 1$). 그러면 $\operatorname{Var}(aX+b) = E[(aX+b - (a\mu+b))^2] = E[a^2(X-\mu)^2] = a^2\operatorname{Var}(X)$. $\square$

정리 · 체비쇼프 부등식 Chebyshev's inequality (호그 §1.9)

분산 하나로 “평균에서 멀리 벗어날 확률”의 상한을 얻는다. 임의의 $k \gt 0$에 대해

$$P\big(|X - \mu| \ge k\sigma\big) \le \frac{1}{k^2}, \qquad\text{동치로}\quad P\big(|X-\mu| \ge \varepsilon\big) \le \frac{\sigma^2}{\varepsilon^2}.$$

분포의 모양을 몰라도 성립하는 보편 부등식이라 대가로 매우 느슨하다 (정규분포의 $2\sigma$ 실제 확률은 약 $0.046$인데 체비쇼프는 $\le 0.25$만 보장). 하지만 이 한 줄이 다음 장의 큰 수의 법칙을 낳는다.

증명 — 분산 적분에서 $|X-\mu|\ge k\sigma$인 영역만 남기고 그 위에서 $(X-\mu)^2 \ge k^2\sigma^2$로 아래를 갈아끼운다.

연속형으로 쓰면 ($A = \{x : |x-\mu| \ge k\sigma\}$)

$$\sigma^2 = \int_{-\infty}^{\infty}(x-\mu)^2 f(x)\,dx \ge \int_A (x-\mu)^2 f(x)\,dx \ge k^2\sigma^2 \int_A f(x)\,dx = k^2\sigma^2\, P(|X-\mu|\ge k\sigma).$$

양변을 $k^2\sigma^2$로 나누면 결론. (이산형은 적분을 합으로 바꾸면 동일.) $\square$

예제 · 주사위의 평균과 분산

공정한 주사위 $X$: $E[X] = \tfrac16(1+2+\cdots+6) = \tfrac{21}{6} = 3.5$. $E[X^2] = \tfrac16(1+4+9+16+25+36) = \tfrac{91}{6}$. 따라서

$$\operatorname{Var}(X) = \frac{91}{6} - \left(\frac72\right)^2 = \frac{91}{6} - \frac{49}{4} = \frac{182 - 147}{12} = \frac{35}{12} \approx 2.92.$$

덤으로 $Y = 2X + 1$(눈금을 두 배 하고 1 더하기)이면 $E[Y] = 8$, $\operatorname{Var}(Y) = 4\cdot\tfrac{35}{12} = \tfrac{35}{3}$ — 평균은 변환을 따라가고 분산은 $2^2$만 곱해진다.

3. 적률생성함수 — 분포의 지문

적률 $E[X], E[X^2], E[X^3], \dots$을 하나씩 구하는 대신, 그것들을 통째로 담은 모함수(generating function)를 만들면 편하다. $e^{tX}$의 기댓값을 잡으면, $t$에 대한 미분이 적률을 한 개씩 “뽑아내는” 기계가 된다.

정의 · 적률생성함수 moment generating function (호그 §1.9)

적률생성함수(MGF)는

$$M(t) = E\big[e^{tX}\big] = \sum_x e^{tx}p(x) \quad\text{또는}\quad \int_{-\infty}^{\infty} e^{tx} f(x)\,dx$$

이며, $0$을 포함하는 어떤 열린구간 $(-h, h)$에서 유한할 때 “존재한다”고 한다 (이 조건이 아래 세 성질의 전제다). 항상 $M(0) = 1$.

① 적률을 생성한다. $e^{tx} = \sum_k \tfrac{(tx)^k}{k!}$를 대입하고 기댓값과 교환하면 $M(t) = \sum_k E[X^k]\,\tfrac{t^k}{k!}$. 따라서 $t=0$에서의 미분이 적률이다:

$$M^{(k)}(0) = E[X^k]. \qquad (\text{특히 } M'(0)=\mu,\ M''(0)=E[X^2].)$$

② 유일성. MGF가 (0의 근방에서) 존재하면 분포를 유일하게 결정한다 — 두 확률변수의 MGF가 일치하면 같은 분포다. MGF는 분포의 지문이다.

③ 합의 MGF = MGF의 곱. $X, Y$가 독립이면

$$M_{X+Y}(t) = M_X(t)\,M_Y(t).$$

(독립이면 $E[e^{t(X+Y)}] = E[e^{tX}]E[e^{tY}]$.) 유일성과 결합하면 — “독립 정규 + 정규 = 정규”, “독립 포아송의 합 = 포아송” 같은 정리를 곱셈 한 번으로 증명하는 강력한 도구가 된다. 3장에서 본격적으로 쓴다.

예제 · MGF로 평균·분산 뽑기

(a) 베르누이 $\mathrm{Ber}(p)$. $M(t) = (1-p)e^{0} + p\,e^{t} = 1 - p + pe^{t}$. $M'(t) = pe^t$이므로 $E[X] = M'(0) = p$; $M''(t) = pe^t$이므로 $E[X^2] = p$, $\operatorname{Var}(X) = p - p^2 = p(1-p)$.

(b) 지수분포 $\mathrm{Exp}(\lambda)$. $M(t) = \int_0^\infty e^{tx}\lambda e^{-\lambda x}dx = \dfrac{\lambda}{\lambda - t}$ (단 $t \lt \lambda$ — 여기서만 존재!). $M'(t) = \lambda/(\lambda-t)^2$이므로 $E[X] = M'(0) = 1/\lambda$, 이어서 $\operatorname{Var}(X) = 1/\lambda^2$. “$t \lt \lambda$”라는 존재 범위 자체가 분포의 꼬리 정보를 담고 있다.

4. 확률변수의 변환 — 근이 아니라 분포를 옮긴다

$X$의 분포를 알 때 $Y = g(X)$의 분포를 구하는 문제다. 이산형은 값을 그러모으면 그만이지만, 연속형은 밀도가 압축·팽창되는 만큼을 야코비안으로 보정해야 한다. 발상은 언제나 같다 — 먼저 CDF를 옮기고, 미분해 pdf를 얻는다.

정리 · 변환 공식 transformation of variables (호그 §1.7)

이산형: $\displaystyle p_Y(y) = \sum_{x:\,g(x)=y} p_X(x)$ (같은 값으로 가는 질량을 모은다).

연속형, $g$가 단조(가역): CDF 방법이 핵심이다. $g$가 순증가면

$$F_Y(y) = P(g(X) \le y) = P\big(X \le g^{-1}(y)\big) = F_X\big(g^{-1}(y)\big),$$

양변을 $y$로 미분하면 야코비안 공식 ($x = g^{-1}(y)$):

$$f_Y(y) = f_X\big(g^{-1}(y)\big)\left|\frac{d}{dy}\,g^{-1}(y)\right| = \frac{f_X(x)}{|g'(x)|}.$$

절댓값은 순감소인 경우($g^{-1}$의 도함수가 음수)까지 한꺼번에 처리한다. $g$가 단조가 아니면(예: $Y = X^2$) 정의역을 단조 조각으로 쪼개 각 가지의 기여를 더한다.

예제 · 변환 세 장면

(a) 선형변환 $Y = aX + b$ ($a \neq 0$). $g^{-1}(y) = (y-b)/a$, $|d g^{-1}/dy| = 1/|a|$이므로 $f_Y(y) = \tfrac{1}{|a|}f_X\!\big(\tfrac{y-b}{a}\big)$. 이것이 정규분포의 표준화·역표준화의 계산적 근거다.

(b) $Y = X^2$, $X \sim U(-1,1)$. 두 가지 $x = \pm\sqrt{y}$가 같은 $y$로 간다. $f_X = \tfrac12$이므로 두 가지를 더해 $f_Y(y) = 2 \cdot \tfrac12 \cdot \tfrac{1}{2\sqrt y} = \tfrac{1}{2\sqrt y}$ ($0 \lt y \lt 1$). 비단조 변환은 “가지 세기”가 관건.

(c) 확률적분변환. $X$가 연속이고 CDF가 $F$면 $U = F(X) \sim U(0,1)$. 거꾸로 $U \sim U(0,1)$에 $F^{-1}$을 씌우면 원하는 $F$를 가진 표본이 나온다 — 난수 시뮬레이션의 기본 원리(역변환 샘플링). (b)의 $Y=-\ln X$ 유형은 연습문제에서 다룬다.

흔한 오해

① “pdf는 확률이다” — 아니다. $f(x)$는 밀도이지 확률이 아니라 $1$을 넘어도 된다. 예: $U(0,\tfrac12)$은 $f = 2$. 확률은 언제나 넓이 $\int f$이지 높이 $f(x)$가 아니다. $P(X = x) = f(x)\,dx$가 아니라 $0$이다.

② 연속형에서 $P(X = a) = 0$. 따라서 $\lt$와 $\le$를 바꿔도 확률이 같다 — 하지만 이산형에서는 절대 아니다(도약 폭만큼 차이). 문제를 풀 때 “연속인가 이산인가”를 먼저 못 박아야 한다.

③ 기댓값·MGF는 존재하지 않을 수 있다. 코시분포 $f(x) = \tfrac{1}{\pi(1+x^2)}$는 $\int |x| f\,dx = \infty$라 평균조차 없고, $M(t) = E[e^{tX}]$는 $t \neq 0$에서 발산해 MGF가 존재하지 않는다. “적률이 다 있으면 MGF가 있다”도 거짓 — 로그정규분포는 모든 적률이 유한하지만 MGF가 $0$의 근방에서 발산한다. 그래서 유일성 정리는 “MGF가 $0$ 근방에서 존재할 때”라는 단서를 반드시 단다.

한 장 요약

2장의 줄거리

확률변수 $X:\Omega\to\mathbb{R}$ = 우연을 수직선으로 옮기는 번역기 → 분포함수 $F(x)=P(X\le x)$ 하나가 모든 것을 결정(단조·우연속·양끝 $0,1$; 이산은 pmf·계단, 연속은 pdf·넓이, 점질량 $=$ 도약) → 기댓값과 분산: 무게중심 $E[X]$와 퍼짐 $\operatorname{Var}=E[X^2]-\mu^2$, 선형성 $E[aX+b]=aE[X]+b$·$\operatorname{Var}(aX+b)=a^2\operatorname{Var}(X)$, 체비쇼프로 꼬리 제어 → 적률생성함수 $M(t)=E[e^{tX}]$: $M^{(k)}(0)=E[X^k]$로 적률을 뽑고, 분포를 유일하게 결정하며, 독립합의 MGF는 곱 → 변환 $Y=g(X)$는 CDF를 옮기고 미분(야코비안).

다음 장 예고: 변수를 둘 이상으로 늘리면 결합분포·주변분포·조건부분포와 공분산·상관이 등장한다. MGF의 “독립합=곱” 성질이 그때 중심극한정리의 엔진이 된다. 임용 최중요 축은 분포 세우기 → 기댓값·확률 계산 → MGF로 분포 식별이다.

연습문제

개념 확인 1 ●●○

이산확률변수 $X$의 pmf가 $p(x) = cx$ ($x = 1, 2, 3, 4$; 그 외 $0$)이다. (1) 상수 $c$를 구하라. (2) $F(x)$를 서술하고 $P(X \le 2)$를 구하라. (3) $E[X]$와 $\operatorname{Var}(X)$를 구하라.

힌트

(1) $\sum p(x) = 1$: $c(1+2+3+4) = 1$. (3) $\operatorname{Var} = E[X^2] - \mu^2$.

풀이

(1) $c(1+2+3+4) = 10c = 1 \Rightarrow c = \tfrac1{10}$.

(2) $p(1)=\tfrac1{10}, p(2)=\tfrac2{10}, p(3)=\tfrac3{10}, p(4)=\tfrac4{10}$. $F$는 각 정수에서 그만큼 뛰는 계단함수이며 $P(X \le 2) = F(2) = \tfrac{3}{10}$.

(3) $E[X] = \tfrac1{10}(1+4+9+16) = \tfrac{30}{10} = 3$. $E[X^2] = \tfrac1{10}(1+8+27+64) = \tfrac{100}{10} = 10$. $\operatorname{Var}(X) = 10 - 3^2 = 1$.

개념 확인 2 ●●○

연속확률변수 $X$의 pdf가 $f(x) = cx^2$ ($0 \lt x \lt 1$; 그 외 $0$)이다. (1) $c$를 구하라. (2) $c \gt 1$임을 확인하고 “pdf $\le 1$”이 필요 없음을 설명하라. (3) $P\!\left(X \gt \tfrac12\right)$와 $E[X]$를 구하라.

힌트

(1) $\int_0^1 cx^2\,dx = 1$. (2) 제약은 $f \ge 0$과 넓이 $=1$뿐 — 높이 상한은 없다.

풀이

(1) $\int_0^1 cx^2\,dx = c/3 = 1 \Rightarrow c = 3$. 즉 $f(x) = 3x^2$.

(2) $f$는 $x \to 1^-$에서 $3 \gt 1$까지 올라간다. pdf에 걸린 조건은 $f \ge 0$과 $\int f = 1$뿐이므로 높이는 얼마든 클 수 있다 — 확률은 높이가 아니라 넓이이기 때문. (밀도가 좁은 구간에 몰리면 값이 커진다.)

(3) $P(X \gt \tfrac12) = \int_{1/2}^1 3x^2\,dx = \big[x^3\big]_{1/2}^1 = 1 - \tfrac18 = \tfrac78$. $E[X] = \int_0^1 x\cdot 3x^2\,dx = \int_0^1 3x^3\,dx = \tfrac34$.

Hogg §1.9 연습 유형 ●●○

확률변수 $X$의 적률생성함수가 $M(t) = \dfrac{1}{3}e^{t} + \dfrac{2}{3}e^{2t}$일 때, (1) $X$의 분포(pmf)를 식별하고 (2) $E[X]$와 $\operatorname{Var}(X)$를 두 방법 — 정의와 $M$의 미분 — 으로 각각 구해 일치를 확인하라.

힌트

$M(t) = \sum_x e^{tx}p(x)$의 꼴과 직접 비교하라. $e^{tx}$의 계수가 $p(x)$. 유일성 정리가 “식별”을 정당화한다.

풀이

(1) $M(t) = e^{t\cdot 1}\!\cdot\tfrac13 + e^{t\cdot 2}\!\cdot\tfrac23$이므로 $P(X=1) = \tfrac13$, $P(X=2) = \tfrac23$ (다른 값은 확률 $0$). MGF 유일성으로 이것이 유일한 분포다.

(2) 정의: $E[X] = 1\cdot\tfrac13 + 2\cdot\tfrac23 = \tfrac53$, $E[X^2] = 1\cdot\tfrac13 + 4\cdot\tfrac23 = 3$, $\operatorname{Var} = 3 - \tfrac{25}{9} = \tfrac{2}{9}$.

미분: $M'(t) = \tfrac13 e^t + \tfrac43 e^{2t}$, $M'(0) = \tfrac13+\tfrac43 = \tfrac53$ ✓. $M''(t) = \tfrac13 e^t + \tfrac83 e^{2t}$, $M''(0) = \tfrac13 + \tfrac83 = 3 = E[X^2]$, $\operatorname{Var} = 3 - (\tfrac53)^2 = \tfrac29$ ✓.

Hogg §1.7 연습 유형 ●●●

$X \sim U(0,1)$이고 $Y = -\ln X$일 때 $Y$의 pdf를 구하고, 어떤 분포인지 밝혀라. 이어서 $E[Y]$를 구하라.

힌트

$g(x) = -\ln x$는 $(0,1)$에서 순감소·가역. $y = -\ln x \Rightarrow x = e^{-y}$, $y$의 범위는 $(0, \infty)$. 야코비안 공식.

풀이

CDF 방법: $F_Y(y) = P(-\ln X \le y) = P(X \ge e^{-y}) = 1 - e^{-y}$ ($y \gt 0$; $X\sim U(0,1)$이라 $P(X \ge a) = 1-a$). 미분하면

$$f_Y(y) = e^{-y} \quad (y \gt 0).$$

즉 $Y \sim \mathrm{Exp}(1)$ (모수 $\lambda = 1$인 지수분포). 야코비안으로 확인: $x = e^{-y}$, $|dx/dy| = e^{-y}$, $f_X = 1$이므로 $f_Y = 1\cdot e^{-y}$ — 일치.

$E[Y] = \int_0^\infty y\,e^{-y}\,dy = 1$ (감마적분 $\Gamma(2)=1!$). 지수분포 $\mathrm{Exp}(1)$의 평균 $1/\lambda = 1$과 맞는다. — 이것이 균등난수로 지수분포를 뽑는 역변환 샘플링이다.

임용 스타일 ●●○ 선형성 · 체비쇼프

어떤 부품 수명 $X$의 평균이 $\mu = 50$(시간), 표준편차가 $\sigma = 5$이다. (1) $Y = \tfrac{X - 50}{5}$의 평균과 분산을 구하라. (2) 체비쇼프 부등식으로 $P(35 \lt X \lt 65)$의 하한을 제시하라.

힌트

(1) 표준화. (2) $35 \lt X \lt 65 \iff |X - 50| \lt 15 = 3\sigma$. 여사건에 체비쇼프.

풀이

(1) $E[Y] = \tfrac{1}{5}(E[X] - 50) = 0$, $\operatorname{Var}(Y) = \tfrac{1}{5^2}\operatorname{Var}(X) = \tfrac{25}{25} = 1$. 표준화는 언제나 평균 $0$·분산 $1$을 만든다.

(2) $|X - 50| \lt 15$은 $k\sigma$에서 $k = 15/5 = 3$. 체비쇼프: $P(|X-50| \ge 15) \le \tfrac{1}{3^2} = \tfrac19$. 여사건을 취해

$$P(35 \lt X \lt 65) = P(|X-50| \lt 15) \ge 1 - \tfrac19 = \tfrac89 \approx 0.889.$$

(분포 모양을 전혀 몰라도 얻는 보장 — 실제 분포가 정규라면 훨씬 큰 $0.997$이지만, 체비쇼프는 “최소한”만 말한다.)

임용 스타일 ●●● 분포 세우기 · MGF 식별

연속확률변수 $X$의 pdf가 $f(x) = \tfrac12 e^{-|x|}$ ($x \in \mathbb{R}$; 라플라스 분포)이다. (1) $f$가 pdf임을 확인하라. (2) 대칭성으로 $E[X]$를 구하라. (3) MGF $M(t)$를 구하고 그 존재 범위를 밝힌 뒤, $M''(0)$으로 $\operatorname{Var}(X)$를 구하라.

힌트

(1),(3) $|x|$ 때문에 $x \lt 0$과 $x \ge 0$으로 나눠 적분한다. (2) $xf(x)$가 기함수. (3) $\int_0^\infty e^{(t-1)x}dx$는 $t \lt 1$에서만 수렴.

풀이

(1) $f \ge 0$이고 $\int_{-\infty}^{\infty}\tfrac12 e^{-|x|}dx = 2\cdot\int_0^\infty \tfrac12 e^{-x}dx = 1$ (대칭). ✓

(2) $x f(x) = \tfrac12 x e^{-|x|}$는 기함수이고 절대적분가능하므로 $E[X] = 0$.

(3) $\displaystyle M(t) = \int_{-\infty}^{0}\!\tfrac12 e^{tx}e^{x}dx + \int_{0}^{\infty}\!\tfrac12 e^{tx}e^{-x}dx = \tfrac12\!\int_{-\infty}^{0}\! e^{(t+1)x}dx + \tfrac12\!\int_0^\infty\! e^{(t-1)x}dx.$ 첫 적분은 $t+1 \gt 0$, 둘째는 $t-1 \lt 0$에서 수렴하므로 존재 범위는 $-1 \lt t \lt 1$이고

$$M(t) = \frac12\cdot\frac{1}{t+1} + \frac12\cdot\frac{-1}{t-1} = \frac{1}{2(t+1)} + \frac{1}{2(1-t)} = \frac{1}{1 - t^2}.$$

$M(t) = (1-t^2)^{-1}$. $M'(t) = 2t(1-t^2)^{-2}$, $M'(0) = 0 = E[X]$ (2와 일치). $M''(t) = 2(1-t^2)^{-2} + 8t^2(1-t^2)^{-3}$, $M''(0) = 2 = E[X^2]$. 따라서 $\operatorname{Var}(X) = 2 - 0^2 = 2$. (존재 범위가 유한한 $(-1,1)$이라는 점 자체가 라플라스 분포의 지수적 꼬리를 반영한다.)

※ 임용 스타일 배지는 기출 유형을 따른 자체 제작 문제입니다.
