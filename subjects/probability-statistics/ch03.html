확률과 통계 · Hogg (§2)

3 다변수 분포와 상관

“두 확률변수가 한 실험에서 함께 태어나면, 각자의 분포만으로는 결코 알 수 없는 것이 생긴다 — 그들이 서로를 어떻게 끌고 다니는가.” 하나의 표본공간 위에 여러 확률변수가 동거하기 시작하는 순간, 확률론의 문법이 바뀐다.

2장까지는 확률변수 하나를 홀로 다뤘습니다. 하지만 현실의 질문은 대개 둘 이상입니다 — 키와 몸무게, 광고비와 매출, 오늘의 주가와 내일의 주가. 이때 결정적인 정보는 각자의 분포가 아니라 둘이 함께 움직이는 방식에 있습니다. 이 장은 그 “함께”를 수학으로 붙잡습니다: 결합분포로 동거를 기술하고($f(x,y)$), 주변·조건부로 시선을 나누고, 공분산과 상관계수로 “같이 움직이는 정도”를 하나의 숫자로 압축합니다. 그리고 이 장의 가장 미묘한 교훈 하나 — 독립이면 무상관이지만, 무상관이라고 독립은 아니다 — 를 반례로 못 박습니다.

1. 결합 · 주변 · 조건부 분포 — 하나의 확률공간, 여러 개의 눈

직관. 확률변수 $X, Y$가 같은 실험에서 나온다면, “$X$가 대략 이 값이고 동시에 $Y$가 대략 저 값일 확률”을 물을 수 있습니다. 이 이차원 확률의 지형도가 결합분포입니다. 결합분포를 알면 나머지는 전부 파생됩니다: 한 변수를 “합쳐 없애면” 주변분포, 한 변수를 “값으로 고정하면” 조건부분포.

정의 · 결합 · 주변 · 조건부 분포 joint, marginal, conditional distribution (호그 §2)

결합분포. 두 확률변수 $(X, Y)$의 결합 pmf(이산)/pdf(연속) $f(x,y)$는 각각

$$P(X = x,\, Y = y) = f(x,y) \quad(\text{이산}), \qquad P\big((X,Y) \in A\big) = \iint_A f(x,y)\,dx\,dy \quad(\text{연속})$$

을 준다. 항상 $f \ge 0$이고 전체 합/적분은 $1$이다.

주변분포 marginal. 관심 없는 변수를 “합쳐 없앤” 것 — 한 변수만의 분포로 되돌아간다:

$$f_X(x) = \sum_{y} f(x,y) \quad(\text{이산}), \qquad f_X(x) = \int_{-\infty}^{\infty} f(x,y)\,dy \quad(\text{연속}).$$

조건부분포 conditional. “$X = x$임을 알았을 때” $Y$의 분포 — 조건부확률 $P(B\mid A) = P(A\cap B)/P(A)$의 밀도 버전:

$$f(y \mid x) = \frac{f(x,y)}{f_X(x)} \qquad (f_X(x) > 0).$$

고정된 $x$에 대해 $f(y\mid x)$는 $y$에 관한 어엿한 분포다 ($\ge 0$, 적분 $=1$).

독립 independence. $X$와 $Y$가 독립이라 함은 결합분포가 주변분포들의 곱으로 완전히 쪼개지는 것:

$$f(x,y) = f_X(x)\, f_Y(y) \quad(\text{모든 } x, y).$$

동치로 $f(y\mid x) = f_Y(y)$ — “$X$를 알아도 $Y$에 대한 앎이 전혀 늘지 않는다”.

예제 · 결합에서 주변·조건부·독립까지

(a) 연속. $f(x,y) = x + y$, $0 < x,y < 1$ (그 밖은 $0$). 주변: $f_X(x) = \int_0^1 (x+y)\,dy = x + \tfrac12$. 조건부: $f(y\mid x) = \dfrac{x+y}{x + \frac12}$ — 이것이 $y$만의 함수로 안 쪼개지므로 $X, Y$는 종속이다.

(b) 곱꼴이면 독립. $f(x,y) = 4xy$, $0독립. 판정 요령: 결합밀도가 “($x$만의 함수)$\times$($y$만의 함수)”로 인수분해되고 정의역이 사각형(변수끼리 얽히지 않음)이면 독립이다.

(c) 정의역이 배신하는 경우. $f(x,y) = 8xy$, $0 < x < y < 1$. 밀도 자체는 $8xy$로 곱꼴처럼 보이지만 정의역 $x 독립이 깨진다 — 큰 $x$는 큰 $y$를 강제한다. 반드시 지지집합의 모양까지 확인해야 한다.

한눈에 — 결합이 원본, 나머지는 그림자

결합분포 $f(x,y)$는 완전한 정보다. 주변은 “한 축으로 눌러 납작하게 만든 그림자”(정보 손실!), 조건부는 “한 좌표에서 자른 단면을 다시 넓이 1로 정규화한 것”. 그래서 주변 두 개를 알아도 결합을 복원할 수 없다 — 같은 주변을 갖는 결합분포가 무수히 많기 때문. 단, 독립일 때만 주변들의 곱이 결합을 되살린다.

2. 기댓값 · 공분산 · 상관 — 함께 움직이는 정도를 숫자 하나로

직관. $X$가 평균보다 클 때 $Y$도 대체로 평균보다 크다면 둘은 “같은 방향”, 반대라면 “엇갈리는 방향”. 이 경향의 부호와 크기를 재는 것이 공분산이고, 그것을 각자의 산포로 나눠 $[-1,1]$ 안에 가둔 무단위 지표가 상관계수입니다.

정의 · 공분산과 상관계수 covariance and correlation (호그 §2)

결합 기댓값. 함수 $g(X,Y)$에 대해

$$E[g(X,Y)] = \sum_x \sum_y g(x,y)\, f(x,y) \quad\text{또는}\quad \iint g(x,y)\, f(x,y)\, dx\, dy.$$

기댓값의 선형성(독립 불필요!). 어떤 경우에도

$$E[aX + bY + c] = a\,E[X] + b\,E[Y] + c.$$

곱의 기댓값(독립일 때만). $X \perp Y$이면 $E[XY] = E[X]\,E[Y]$.

공분산.

$$\operatorname{Cov}(X,Y) = E\big[(X - \mu_X)(Y - \mu_Y)\big] = E[XY] - E[X]\,E[Y].$$

부호가 방향을, 크기가 정도를 (단위에 얽혀) 나타낸다. $\operatorname{Cov}(X,X) = \operatorname{Var}(X)$.

상관계수. 공분산을 표준편차 곱으로 정규화한 무단위 지표:

$$\rho = \rho_{XY} = \frac{\operatorname{Cov}(X,Y)}{\sigma_X\, \sigma_Y} \in [-1, 1].$$

분산의 합 공식.

$$\operatorname{Var}(X + Y) = \operatorname{Var}(X) + \operatorname{Var}(Y) + 2\operatorname{Cov}(X,Y).$$

독립(또는 무상관)이면 마지막 항이 사라져 분산이 그냥 더해진다.

왜 $\rho \in [-1,1]$인가 — 코시-슈바르츠의 확률 버전. 등호는 완전한 선형관계일 때뿐.

중심화한 변수 $U = X - \mu_X$, $V = Y - \mu_Y$를 놓으면 $\operatorname{Cov}(X,Y) = E[UV]$, $\sigma_X^2 = E[U^2]$, $\sigma_Y^2 = E[V^2]$. 임의의 실수 $t$에 대해 $E[(U - tV)^2] \ge 0$ (제곱의 기댓값). 전개하면

$$E[U^2] - 2t\,E[UV] + t^2 E[V^2] \ge 0 \quad(\forall t).$$

$t$에 관한 이차식이 항상 비음이므로 판별식 $\le 0$: $(E[UV])^2 \le E[U^2]\,E[V^2]$, 즉 $\operatorname{Cov}(X,Y)^2 \le \sigma_X^2\sigma_Y^2$. 양변을 $\sigma_X^2\sigma_Y^2$로 나누면 $\rho^2 \le 1$.

등호는 어떤 $t$에서 $E[(U - tV)^2] = 0$일 때, 곧 $U = tV$가 확률 1로 성립할 때 — $X$와 $Y$가 완벽한 직선관계 $X - \mu_X = t(Y - \mu_Y)$를 이룰 때다. $\rho = +1$은 기울기 양(우상향 직선), $\rho = -1$은 기울기 음.

예제 · 공분산·상관 계산과 “선형성은 언제나”

(a) $\operatorname{Var}(X-Y) = \operatorname{Var}(X) + \operatorname{Var}(Y) - 2\operatorname{Cov}(X,Y)$. 뺄셈에서는 부호가 바뀐다 — 독립이면 차의 분산도 합의 분산과 같아진다 ($\operatorname{Var}X + \operatorname{Var}Y$).

(b) 선형성은 독립을 요구하지 않는다. 주사위 두 개의 눈 $X_1, X_2$ 가 어떻게 얽혀 있든 $E[X_1 + X_2] = E[X_1] + E[X_2] = 3.5 + 3.5 = 7$. 반면 $\operatorname{Var}(X_1 + X_2)$는 공분산 항 때문에 얽힘에 의존한다.

(c) 표준화. $\rho$는 단위·평행이동·양의 스케일에 불변: $\rho_{aX+b,\,cY+d} = \operatorname{sgn}(ac)\,\rho_{XY}$. 그래서 키를 cm로 재든 m로 재든 상관계수는 그대로다.

3. 조건부기댓값 — 부분정보로 예측하고, 분산을 쪼개기

직관. “$X$를 이미 안다”면 $Y$에 대한 최선의 예측은 조건부분포의 평균 $E[Y\mid X]$입니다. 그런데 $X$ 자체가 확률변수이므로 $E[Y\mid X]$도 $X$의 함수인 확률변수입니다 — 이 관점이 반복기댓값과 분산분해라는 두 강력한 도구를 낳습니다.

정의 + 정리 · 조건부기댓값과 반복기댓값 conditional expectation, tower rule (호그 §2)

조건부기댓값. $X = x$가 주어졌을 때 $Y$의 평균:

$$E[Y \mid X = x] = \sum_y y\, f(y\mid x) \quad\text{또는}\quad \int y\, f(y\mid x)\, dy.$$

이를 $x$의 함수로 보고 $x$에 $X$를 되꽂으면, 확률변수 $E[Y\mid X]$를 얻는다.

반복기댓값 법칙(전체기댓값 법칙, 호그 §2).

$$E\big[\,E[Y \mid X]\,\big] = E[Y].$$

“조건부 평균들을 다시 $X$에 대해 평균 내면 전체 평균이 복원된다.”

분산분해(전체분산 법칙, 호그 §2).

$$\operatorname{Var}(Y) = \underbrace{E\big[\operatorname{Var}(Y\mid X)\big]}_{\text{집단 안 산포}} + \underbrace{\operatorname{Var}\big(E[Y\mid X]\big)}_{\text{집단 간 산포}}.$$

전체 변동 = “각 조건 안에서의 평균 변동” + “조건별 평균값들 사이의 변동”.

반복기댓값의 한 줄 증명 — 결합을 조건부×주변으로 쪼갠 뒤 적분 순서 바꾸기.

$f(x,y) = f(y\mid x)\, f_X(x)$이므로 (연속으로 쓰면)

$$E\big[E[Y\mid X]\big] = \int \Big(\underbrace{\int y\, f(y\mid x)\,dy}_{=\,E[Y\mid X=x]}\Big) f_X(x)\, dx = \iint y\, f(y\mid x) f_X(x)\, dy\, dx = \iint y\, f(x,y)\, dy\, dx = E[Y].$$

이산도 합으로 똑같다. 분산분해는 $\operatorname{Var}(Y) = E[Y^2] - (E[Y])^2$에 같은 기법을 두 번 적용하면 나온다.

예제 · 무작위 개수의 합 (복합분포)

하루 손님 수 $N$은 평균 $\lambda$, 각 손님의 지출 $Y_i$는 평균 $\mu$·분산 $\sigma^2$로 서로 독립이고 $N$과도 독립. 총매출 $S = \sum_{i=1}^{N} Y_i$의 평균은 반복기댓값으로 즉시:

$$E[S] = E\big[E[S\mid N]\big] = E[N\mu] = \mu\, E[N] = \mu\lambda.$$

분산분해로 $\operatorname{Var}(S) = E[N]\sigma^2 + \mu^2 \operatorname{Var}(N)$ — “손님 수의 불확실성”이 매출 분산에 어떻게 기여하는지가 두 번째 항에 분리되어 드러난다. $N$을 안다고 가정하는 순간 문제가 쉬워지는 것이 조건부의 힘이다.

4. 변수변환과 합 — 야코비, 합성곱, 적률생성함수

직관. $X, Y$의 결합분포를 알 때 $U = g_1(X,Y)$, $V = g_2(X,Y)$의 분포는 어떻게 될까요? 일변수의 치환적분이 다변수에서 야코비 행렬식 으로 일반화됩니다. 특히 독립 확률변수의 합은 합성곱이라는 깔끔한 공식을 갖고, 적률생성함수는 그 합성곱을 곱셈으로 바꿔 버립니다.

정의 + 정리 · 변수변환·합성곱·MGF change of variables, convolution, MGF (호그 §2)

야코비 변환. $(X,Y) \mapsto (U,V)$가 일대일이고 역변환 $x = x(u,v),\, y = y(u,v)$가 미분가능이면

$$f_{U,V}(u,v) = f_{X,Y}\big(x(u,v),\, y(u,v)\big)\; \big|J\big|, \qquad J = \det\!\begin{pmatrix} \partial x/\partial u & \partial x/\partial v \\[2pt] \partial y/\partial u & \partial y/\partial v \end{pmatrix}.$$

야코비 $|J|$는 “부피(넓이)가 늘어난 배율”을 보정한다 — 일변수 $|dx/du|$의 다변수 판.

독립 합의 분포 = 합성곱 convolution. $X \perp Y$이고 $Z = X + Y$이면

$$f_Z(z) = \int_{-\infty}^{\infty} f_X(x)\, f_Y(z - x)\, dx.$$

적률생성함수의 곱. $M_X(t) = E[e^{tX}]$에 대해 $X \perp Y$이면

$$M_{X+Y}(t) = E\big[e^{t(X+Y)}\big] = E[e^{tX}]\,E[e^{tY}] = M_X(t)\, M_Y(t).$$

MGF가 분포를 유일하게 결정하므로(유일성), 이 곱셈이 곧 “어떤 분포족이 합에 대해 닫혀 있는가”를 판별한다.

예제 · MGF가 합성곱을 대신한다

정규분포의 재생성. $X \sim N(\mu_1, \sigma_1^2)$, $Y \sim N(\mu_2, \sigma_2^2)$ 독립이면 $M_X(t) = e^{\mu_1 t + \frac12\sigma_1^2 t^2}$이므로

$$M_{X+Y}(t) = e^{(\mu_1+\mu_2)t + \frac12(\sigma_1^2+\sigma_2^2)t^2}.$$

이것은 다시 정규분포 $N(\mu_1+\mu_2,\, \sigma_1^2+\sigma_2^2)$의 MGF — 적분(합성곱) 한 번 없이 “정규 + 정규 = 정규”가 나온다. 같은 방식으로 포아송끼리의 합은 포아송, 감마(같은 척도)끼리의 합은 감마임이 즉시 따라온다. 합의 분포는 MGF 곱으로 알아보는 것이 정석이다.

흔한 오해

① 독립 ⇒ 무상관은 참, 역은 거짓. $\rho = 0$은 선형 관계가 없다는 뜻일 뿐 — 곡선관계는 얼마든지 있을 수 있다. 반례: $X$가 $\{-1, 0, 1\}$에 균등하고 $Y = X^2$이면 $Y$는 $X$에 완전히 종속인데도 $E[XY] = E[X^3] = 0 = E[X]E[Y]$라 $\operatorname{Cov} = 0$, $\rho = 0$. 무상관이나 독립 아님. (예외: 결합정규에서는 무상관 $\iff$ 독립.)

② $\rho$는 오직 선형관계의 척도. $\rho = 0.9$가 “90% 설명” 같은 뜻이 아니고, $\rho$가 작아도 강한 비선형 종속이 숨을 수 있다. 그림을 안 보고 $\rho$ 하나로 결론짓지 말 것 (앤스컴의 사총사가 경고한다).

③ 공분산 부호만 믿고 크기를 비교하지 말 것. 공분산은 단위에 얽혀 있어(키$\times$몸무게의 단위) 두 쌍의 “강도”를 직접 비교할 수 없다. 비교는 무단위인 $\rho$로 해야 한다.

④ 주변 두 개로 결합을 만들 수 없다. 독립이 아닌 한 $f(x,y) \ne f_X(x)f_Y(y)$이며, 같은 주변을 갖는 결합분포는 무수히 많다.

⑤ $\operatorname{Var}(X+Y) = \operatorname{Var}X + \operatorname{Var}Y$는 독립(무상관)일 때만. 일반적으로 $2\operatorname{Cov}$ 항을 빠뜨리면 오답이다.

한 장 요약

3장의 줄거리

결합분포 $f(x,y)$가 원본 — 합쳐 없애면 주변, 값으로 자르면 조건부, 곱으로 쪼개지면 독립 → 공분산·상관: 함께 움직이는 정도를 $\operatorname{Cov} = E[XY] - E[X]E[Y]$로, 무단위로는 $\rho = \operatorname{Cov}/(\sigma_X\sigma_Y) \in [-1,1]$로 (코시-슈바르츠); 분산의 합은 $2\operatorname{Cov}$만큼 어긋난다 → 조건부기댓값 $E[Y\mid X]$: 반복기댓값 $E[E[Y\mid X]] = E[Y]$과 분산분해가 “부분정보로 나눠 정복”을 가능케 한다 → 변수변환은 야코비, 독립 합은 합성곱, 그리고 MGF의 곱이 합의 분포를 곱셈 한 번으로 알려준다.

이 장의 급소: “독립 ⇒ 무상관, 그러나 무상관 ⇏ 독립”과 “선형성은 언제나, 곱과 분산합은 독립일 때만”. 두 문장이 임용·시험에서 가장 자주 함정으로 나온다.

연습문제

개념 확인 1 ●●○

결합밀도 $f(x,y) = x + y$ ($0 < x, y < 1$, 그 밖은 $0$)에 대하여 (1) 주변밀도 $f_X, f_Y$를 구하고 (2) $X, Y$의 독립 여부를 판정하며 (3) 조건부밀도 $f(y \mid x)$를 구하라.

힌트

주변은 한 변수로 적분. 독립은 $f(x,y) \overset{?}{=} f_X(x)f_Y(y)$ 검사.

풀이

(1) $f_X(x) = \int_0^1 (x+y)\,dy = x + \tfrac12$ ($0

(2) $f_X(x)f_Y(y) = (x+\tfrac12)(y+\tfrac12) = xy + \tfrac12 x + \tfrac12 y + \tfrac14 \ne x + y = f(x,y)$ — 종속이다.

(3) $f(y\mid x) = \dfrac{x+y}{x+\frac12}$ ($0

개념 확인 2 ●●○

$f(x,y) = 4xy$ ($0

힌트

곱꼴 + 사각형 정의역 → 독립. 독립이면 $E[XY] = E[X]E[Y]$이고 곧 $\operatorname{Cov} = 0$.

풀이

(1) $f_X(x) = \int_0^1 4xy\,dy = 2x$, $f_Y(y) = 2y$이고 $4xy = (2x)(2y) = f_X f_Y$, 정의역도 사각형 — 독립.

(2) $E[X] = \int_0^1 x\cdot 2x\,dx = \tfrac23$ (대칭으로 $E[Y] = \tfrac23$). 독립이므로 $E[XY] = E[X]E[Y] = \tfrac49$. 따라서 $\operatorname{Cov}(X,Y) = \tfrac49 - \tfrac49 = 0$, $\rho = 0$. (독립이니 무상관은 당연 — 방향이 옳다.)

Hogg §2 연습 유형 ●●●

무상관인데 독립이 아닌 반례. $X$가 $\{-1, 0, 1\}$에 각 확률 $\tfrac13$로 균등하고 $Y = X^2$이라 하자. (1) $\operatorname{Cov}(X,Y) = 0$임을 보이고 (2) 그럼에도 $X, Y$가 독립이 아님을 보여라.

힌트

$E[XY] = E[X^3]$. 독립이 깨짐은 결합확률 하나가 주변곱과 다름을 보이면 된다.

풀이

(1) $E[X] = 0$ (대칭). $E[XY] = E[X\cdot X^2] = E[X^3] = \tfrac13(-1) + \tfrac13(0) + \tfrac13(1) = 0$. 따라서 $\operatorname{Cov}(X,Y) = E[XY] - E[X]E[Y] = 0 - 0 = 0$ — 무상관.

(2) $P(X = 0) = \tfrac13$, $P(Y = 0) = P(X = 0) = \tfrac13$. 그런데 $P(X = 0,\, Y = 1)$을 보자: $Y = 1 \iff X = \pm 1$이므로 $X = 0$과 $Y = 1$은 동시에 불가능 — $P(X=0, Y=1) = 0$. 반면 $P(X=0)\,P(Y=1) = \tfrac13 \cdot \tfrac23 = \tfrac29 \ne 0$. 결합 $\ne$ 주변곱이므로 독립이 아니다.

교훈: $\rho = 0$은 선형 무관계일 뿐, $Y = X^2$이라는 완벽한 (곡선) 종속과 공존한다.

Hogg §2 연습 유형 ●●●

$\operatorname{Var}(X) = 4$, $\operatorname{Var}(Y) = 9$, $\rho_{XY} = -\tfrac12$일 때 (1) $\operatorname{Cov}(X,Y)$, (2) $\operatorname{Var}(X + Y)$, (3) $\operatorname{Var}(2X - 3Y)$를 구하라.

힌트

$\operatorname{Cov} = \rho\,\sigma_X\sigma_Y$. 그리고 $\operatorname{Var}(aX+bY) = a^2\operatorname{Var}X + b^2\operatorname{Var}Y + 2ab\operatorname{Cov}(X,Y)$.

풀이

(1) $\sigma_X = 2$, $\sigma_Y = 3$이므로 $\operatorname{Cov}(X,Y) = \rho\,\sigma_X\sigma_Y = -\tfrac12 \cdot 2 \cdot 3 = -3$.

(2) $\operatorname{Var}(X+Y) = 4 + 9 + 2(-3) = 7$.

(3) $\operatorname{Var}(2X - 3Y) = 4\cdot4 + 9\cdot9 + 2(2)(-3)\operatorname{Cov} = 16 + 81 + 2(2)(-3)(-3) = 16 + 81 + 36 = 133$. (교차항 계수 $2ab = 2(2)(-3) = -12$에 $\operatorname{Cov} = -3$을 곱해 $+36$.)

임용 스타일 ●●● 합성곱 · MGF

$X, Y$가 독립이고 각각 모수 $\lambda$인 지수분포 ($f(t) = \lambda e^{-\lambda t}$, $t>0$)를 따른다. $Z = X + Y$의 밀도를 (1) 합성곱으로, (2) MGF로 구하고 두 결과가 일치함을 확인하라.

힌트

합성곱: $f_Z(z) = \int_0^z f_X(x)f_Y(z-x)\,dx$ (지지집합 $0

풀이

(1) $z > 0$에서, 피적분이 $0$이 아니려면 $0 < x < z$:

$$f_Z(z) = \int_0^z \lambda e^{-\lambda x}\,\lambda e^{-\lambda(z-x)}\,dx = \lambda^2 e^{-\lambda z}\int_0^z dx = \lambda^2 z\, e^{-\lambda z}.$$

이는 형상모수 $2$·비율 $\lambda$인 감마분포 (=에를랑 분포)다.

(2) $M_Z(t) = M_X(t)M_Y(t) = \Big(\dfrac{\lambda}{\lambda - t}\Big)^2$ — 이것이 바로 감마$(k=2, \lambda)$의 MGF. 유일성에 의해 $f_Z(z) = \lambda^2 z\, e^{-\lambda z}$와 일치한다. $\blacksquare$

(서술 요령: 두 방법이 같은 답을 주는 것은 MGF의 유일성 덕분임을 명시하면 만점 답안이다.)

임용 스타일 ●●● 반복기댓값 · 분산분해

동전을 던져 앞면(확률 $\tfrac12$)이면 $Y \sim N(0, 1)$, 뒷면이면 $Y \sim N(3, 1)$에서 값을 뽑는다. 지시변수 $X$($X=1$: 앞, $X=0$: 뒤)를 이용해 (1) $E[Y]$를 반복기댓값으로, (2) $\operatorname{Var}(Y)$를 분산분해로 구하라.

힌트

$E[Y\mid X]$와 $\operatorname{Var}(Y\mid X)$는 각 조건에서 $0/3$, $1/1$. 전체분산 = $E[\operatorname{Var}(Y\mid X)] + \operatorname{Var}(E[Y\mid X])$.

풀이

(1) $E[Y\mid X=1] = 0$, $E[Y\mid X=0] = 3$이므로 $E[Y] = E[E[Y\mid X]] = \tfrac12(0) + \tfrac12(3) = \tfrac32$.

(2) 집단 안 산포: $\operatorname{Var}(Y\mid X) = 1$ (두 경우 모두) 이므로 $E[\operatorname{Var}(Y\mid X)] = 1$. 집단 간 산포: $E[Y\mid X]$는 값 $0$과 $3$을 각 $\tfrac12$로 갖는 확률변수 — 그 분산은 $E[(E[Y\mid X])^2] - (E[Y])^2 = \tfrac12(0) + \tfrac12(9) - (\tfrac32)^2 = 4.5 - 2.25 = 2.25$. 따라서

$$\operatorname{Var}(Y) = 1 + 2.25 = 3.25.$$

“각 정규성분의 내부 분산 1”과 “두 평균 $0, 3$이 벌어져 만든 분산 2.25”가 더해진다 — 혼합분포의 분산이 왜 성분 분산보다 큰지가 분산분해로 명확해진다. $\blacksquare$
